# Data Cleaning & Reporting Automation

## Project Overview
This project automates data cleaning and reporting using Python. It processes raw sales data by handling missing values, removing duplicate records, and generating cleaned datasets and reports.

## Features
- Read sales data from a CSV file
- Handle missing values
- Remove duplicate records
- Save cleaned data
- Generate Excel report
- Generate summary report

## Technologies Used
- Python
- Pandas
- OpenPyXL

## Project Structure

```
Data-Cleaning-Reporting-Automation/
│── main.py
│── sales_data.csv
│── requirements.txt
│── README.md
│── cleaned_data.csv
│── report.xlsx
│── summary.txt
```

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python main.py
```

## Output Files

After running the project, the following files are generated:

- cleaned_data.csv
- report.xlsx
- summary.txt
